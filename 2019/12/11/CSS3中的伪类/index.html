CSS3中的伪类

发表于 2019-12-11 分类于 CSS

动态伪类选择器

    动态伪类选择器包括 :link 是指链接未被访问时，通常用在a标签上去掉下划线和颜色。 :visited 指元素已经被访问过后的状态, :hover 指用户的鼠标停留在元素上, :active 指元素被激活, :focus 指元素获得焦点。

    其中如果是button标签中伪类的执行顺序是hover-active-focus。
如果是在a标签中伪类的执行顺序是hover-active。动态伪类还能替完成很多之前需要js才能完成的动效，比如鼠标停留在某个元素上，元素宽度高度增加等等

目标伪类选择器

    :target一般用在a标签的href里加#ID或#锚点，可以链接到指定的元素。使用target我们可以实现手风琴功能，相册，tab，幻灯片等效果。下面是一个简单的手风琴

UI元素状态伪类选择器

    通常用于form表单中的各种元素包括
:checked指选中状态
:enabled指启用状态
:disabled指不可用状态伪类选择器

伪元素

    CSS3中的伪元素是使用两个冒号来和伪类选择器做区分。包括 ::first-letter 指文本中的第一个字母，::first-line 指文本的第一行文本。 ::before 和 ::after 是我们常用的伪类，生成内容需要content属性，一般用于清除浮动。伪元素还可以用来画三角形，然后使用position来定位，举个例子
